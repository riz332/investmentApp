# Investment App
This app will help user manage their assets and keep track of their portfolio. It can bring together a varied list of investment information together to help with tracking. 

This is just a concept at the moment and the application will be using some free tools to test out the requirements and create a POC

# Tech Stack
## Front End
Front end will be using React using Next.js framework

## Back End
THis will use .Net Core (v10 but upgradable to latest when needed) with EF COre

## Database
PostgresSQL deployed on a docker container
